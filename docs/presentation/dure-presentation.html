What is Dure?

A distributed e-commerce platform that empowers small shop owners with complete control over their online stores.

Built with Rust and egui, it provides a modern solution for running e-commerce operations without relying on traditional centralized server infrastructure.

The Name: "Dure" (두레)

Derived from the Korean word "doorae"
meaning "a community helping each other"

This reflects our core philosophy:

Enabling shop owners to build sustainable businesses through distributed technology and community cooperation

Five Key Components

  • Identity Management - Secure authentication
  • Guest Front (WASM) - Customer experience
  • Store Front (WASM) - Shop display
  • Hosting Management (EGUI) - Infrastructure control
  • Store Management (EGUI) - Business operations

Identity Management

  • Secure authentication using private/public key pairs
  • Firebase/Supabase integration
  • Attestation for WASM/EGUI apps with GitHub Sigstore

Guest & Store Front

Guest Front (WASM)

  • Browse products
  • Manage carts
  • Complete purchases
  • Minimal identity requirements

Store Front (WASM)

  • Display products
  • Manage listings
  • Portone integration
  • KakaoPay integration

Hosting & Store Management

Hosting Management

  • DNS management (octodns)
  • Database setup
  • Site deployment

Store Management

  • Promotions
  • Products & Orders
  • Shipments
  • Accounts
  • Shared listings

Technical Architecture

  • Language: Rust (2021 edition) for performance and reliability
  • UI Framework: egui + eframe with Material3 design
  • Platform Support: Desktop (Linux, Windows, macOS), Android, WASM
  • Database: Diesel ORM with SQLite, optional PostgreSQL
  • Async Runtime: asupersync for efficient I/O operations

Three Deployment Models

🖥️ Desktop

EGUI with system tray

📱 Mobile

Android native-activity

🌐 Web

WASM for guests & stores

Hosting Flexibility

Shop owners can choose their preferred infrastructure:

  • Cloud Platforms: GCP, Firebase, or Supabase
  • DNS Providers: Cloudflare, Porkbun, DuckDNS, Google Cloud DNS
  • Payment Gateways: Portone and KakaoPay integration
  • Self-Hosting: Deploy on cafe24 or custom VPS

Shop owners maintain complete control over their data and infrastructure choices

Key Advantages

💰 Zero Transaction Fees

Unlike traditional platforms that charge 2% or more per transaction, Dure does not charge transaction fees. Shop owners keep their full revenue.

🌐 Distributed Architecture

Each shop owner maintains their own database with content distributed via CDN. Better scalability and reduced infrastructure costs.

🔒 Data Ownership

Complete control over customer data, product information, and business analytics.

Content Delivery

Efficient content delivery through distributed infrastructure

  • Shop owners store content on their own CDN
  • Fast access for customers worldwide
  • Real-time synchronization via websocket connections
  • Instant communication between stores and customers

Security

  • Authentication: Passkey and FIDO2 with ES256 encryption
  • Site-to-Site Communication: ChaCha20-Poly1305 encryption
  • DDoS Protection: fail2ban integration
  • Access Control: Whitelisted host connections
  • Attestation: GitHub Sigstore verification

Customer Support

Leverages modern AI services for efficient customer support

  • Integrate AI assistants
  • Resolve customer inquiries efficiently
  • Streamline communication
  • Reduce operational overhead
  • Maintain quality service
